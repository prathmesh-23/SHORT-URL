This is a [Express.js & Node.js ](https://expressjs.com/) project with [TailwindCSS](https://tailwindcss.com/).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:8001](http://localhost:8001) with your browser to see the result.

You can start editing the page by modifying `/view/index.ejs`. The page auto-updates as you edit the file.
